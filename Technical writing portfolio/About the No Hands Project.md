# About the No Hands Project

Built with holistic accessibility in mind, the No Hands Project is a communication app designed for people who cannot use their device with their hands or direct touch. The No Hands Project lets you send messages, make calls, and interact with their device's UI by speaking certain commands and making customizable gestures.

**Note:** The No Hands Project is only available in the English language.

To get started with the No Hands Project app, refer to the No Hands Project User Guide and Data Policy.

## Configure No Hands Project Settings
You can customize what actions the No Hands Project can detect to meet your needs and preferences. You can also set the app's sensitivity to gestures.
- If you adjust the gesture sensitivity to detect large gestures, this will require more mobility and effort to activate.
- It's recommended to set the gesture sensitivity to the largest size and longest duration that you're comfortable with.

### Customize gestures
1. On the No Hands Project app, select **Settings**.
2. Go to the gesture you want to edit and select **Edit**.
	- **Gesture Sensitivity:**
		- To detect smaller gestures, set the sensitivity to **Low**. This allows gestures like slight head movement to be used.
		- To detect larger gestures, set the sensitivity to **High**. Smaller movements won't be accidentally triggered with this setting.
	- **Ignore Gestures:**
		- **Shorter than:** Ignores gestures that are less than the time you set.
		- **Longer than:** Ignores gestures that are longer than the time you set.
3. To test the gesture sensitivity, select **Test settings**.
4. Once you're done, go back to the main menu.

## Tips to Use the No Hands Project
- Use the No Hands Project app in a well-lit environment to better detect your face and gestures. Make sure that our device's camera is mounted just below eye level.
- Make sure that your device's battery is charged when using the app.
- When using the app, use one gesture at a time so that the app doesn't get confused.

## Troubleshoot No Hands Project Issues 
### The App Doesn't Detect My Gestures
#### Check Your Lighting
Gestures might not get detected by the app when you're in a dimly-lit environment. Make sure to have good lighting so that your device's camera can recognize your face and gestures.
#### Lower Your Gesture Sensitivity
1. On the No Hands Project app, select **Settings**.
2. On the gesture you want to edit, select **Edit**.
3. Set the Gesture Sensitivity to **Low**.
4. To test the new sensitivity setting, select **Test settings**.
#### Check Your Device Placement
Make sure that your face is in view of your device's camera for gesture detection to work.
### Unintended Facial Expressions and Eye Movements Trigger the App
#### Adjust Gesture Settings
1. On the No Hands Project app, select **Settings**.
2. On the gesture you want to edit, select **Edit**.
3. Under "Ignore Gestures," set the "Shorter than" value from 1s to a higher value.
	- This lowers the chance of unintended gestures without compromising the app's detection.
### The "Raise Hand" Gesture Isn't Detected
In order for "Raise Hand" to be detected, make sure that your hand is in an open position, with fingers pointing outward.
