# Learn about Screenshot Restrictions on Work Devices

- **Who:** All (COMPANY) product support agents
- **Regions:** Global
- **Priority:** Medium

Starting on May 2024, all (COMPANY) resources will have screenshot restrictions on their work devices. Screenshots can still be shared with other team members, but they cannot save or copy screenshots to be shared outside of the (COMPANY) file-sharing platform.

You can still upload screenshots to the following tools:
- (COMPANY) Case Manager tool
- (COMPANY)'s file-sharing platform
- (COMPANY)'s communication platform

To support (COMPANY) Data Security Policies, security controls have been implemented that may affect how you share screenshots on company-owned devices. These measures help protect the company's data from accidental leaks.

To learn more, refer to the "New Screenshot Functionality" article.
